# 社保缴费记录查询系统

这是一个基于React + TypeScript + Vite构建的社保缴费记录查询系统前端项目。

## 项目功能

- 用户登录和身份验证
- 个人信息管理
- 社保缴费记录查询
- 缴费记录导出

## 技术栈

- React 18
- TypeScript
- Vite
- React Router
- Redux Toolkit

## 开发环境设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
  ├── assets/      # 静态资源
  ├── components/  # 公共组件
  ├── pages/       # 页面组件
  ├── store/       # Redux状态管理
  └── router.tsx   # 路由配置
```
